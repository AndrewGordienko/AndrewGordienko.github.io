What is a neural network?

You may have heard of neural networks as the new revolutionary idea that will create the Terminator and end us all. You may also be wondering what exactly a neural network is outside of the doomsday predictions, and this is the topic of this article.

Classifications of Artificial Intelligence

Artificial Intelligence encompasses everything related to having code fulfilling a task that a human is able to do. This ranges from Youtube recommendations to tree searches for playing checkers, to a computer learning how to play the game of Go at an elite level.

Machine learning is all artificial intelligence that learns, but not necessarily with neural networks as that is the next subcategory. This seems a little counterintuitive but examples of this are where nodes collect information about the state and then are backpropagated depending on the outcome. For instance, tree searches are a really good example of where board states are assigned a node which is then updated depending on the outcome of playing that board. By doing this the machine is able to learn how to make good moves and does not use neural networks. If you are curious about this tree search I suggest you read up on the Monte Carlo Tree Search.

Deep learning is machine learning that uses neural networks. This is where all the fun stuff exists! From computer vision to natural language processing to teaching AI to get really good at games or tasks, this is the subsection where all this magic resides and has neural networks as the backbone.
